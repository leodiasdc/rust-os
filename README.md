# 🦀 Minimal Rust OS

This repository contains a minimal operating system written in Rust, developed as a learning project by following the excellent blog series:

 **Writing an OS in Rust — Philipp Oppermann**

## 📚 About the Tutorial

This project closely follows the step-by-step tutorial series that builds a small operating system from scratch using Rust.

Each post in the series introduces fundamental OS concepts and provides complete code, making it possible to follow along and understand how things work under the hood.

## ⚠️ Disclaimer

This is a **learning project**, not a production-ready operating system.  
Most of the implementation is based on the tutorial, with possible small modifications and experiments.
